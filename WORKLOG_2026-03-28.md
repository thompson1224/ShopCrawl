## 적용 내용

### 1. XSS 방지

- 핫딜 카드 렌더링을 `innerHTML` 기반 문자열 삽입에서 DOM API 기반 생성으로 변경
- AI 검색 답변도 HTML 직접 주입 대신 텍스트/허용 링크만 렌더링하도록 변경
- 외부 링크는 `http/https`만 허용하고 `noopener noreferrer` 적용

대상 파일:

- `templates/main.js`

### 2. 로그인 토큰 저장 방식 변경

- JWT를 `localStorage`에 저장하던 흐름 제거
- 네이버 로그인 완료 후 `HttpOnly` 쿠키에 액세스 토큰 저장
- `/api/auth/logout` 추가
- `/api/auth/me`는 Authorization 헤더 없이 쿠키 기반으로도 동작하도록 수정

대상 파일:

- `app.py`
- `auth.py`
- `templates/index.html`

### 3. OAuth state 검증 추가

- `/api/auth/naver/login`에서 state를 쿠키에 저장
- `/api/auth/naver/callback`에서 state를 검증 후 불일치 시 차단

대상 파일:

- `app.py`

### 4. 이미지 프록시 SSRF 방지

- `/image-proxy`에 허용 소스별 도메인 allowlist 적용
- `localhost`, loopback, 잘못된 scheme/userinfo 차단
- redirect 차단
- `image/*` 응답만 허용

대상 파일:

- `app.py`
